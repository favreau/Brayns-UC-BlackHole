# Black Hole renderer

![banner](doc/blackhole_banner.png)

## Description
The Black Hole Renderer plugin is a black hole visualizer built on top of [Blue Brain Brayns](https://github.com/BlueBrain/Brayns), the Blue Brain rendering platform.
